A quick and dirty guide to uploading from DSCDecoder

  • Turn Decoding OFF (square "stop" button")
  • Set up the UPD Output
  • Options ... Output ... UDP/IP Output ... Setup
  • Choose your preferred RXID (fewer than 18 characters)





    • Destination IP Address www.yaddnet.org
    Destination Port4530
    Source Identification RXID Freq
    Source Port4531 (not important)





    • Enable UDP Output
    • Options ... Output ... UDP/IP Output ... Enable=tick
    • Turn decoding back on (round Green button)


    THE FREQUENCY MUST BE IN kHz with the appropriate DECIMAL place
    Don't forget to add the "dot zero" where appropriate!

    • 2177.0
    • 2187.5
    • 4207.5
    • 6312.0
    • 8414.5
    • 12577.0
    • 16804.5
    • 156525.0


    Once you are decoding new signals you should see them in the appropriate pages on YaDDNet,
    and also see each incoming packet listed on the DSC Decoder Input Log
